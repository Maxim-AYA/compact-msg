## 8. Риски и наблюдения

1. **Отставание по К1, К2.** Фундаментная плита по К1 только начата (щебёнка 68%), по К2 — ещё не стартовала. Остальные 4 корпуса уже на стенах/перекрытиях. Риск каскадного смещения сроков по этим двум корпусам.
2. **Перевыполнение >100% на гидроизоляции К3, К4, К6** — формальный сигнал о расхождении проектных и фактических объёмов; нужно валидировать у подрядчика (Горошков).
3. **«-» в графе факт-окончание** для нескольких активных работ (К4, К5: расшивка швов, обмазочная ГИ К5, утепление К4, лестницы К5) — фактический срок завершения не зафиксирован, риск скрытой просрочки.
4. **Длинный «хвост» на 86,9% работ ещё не начат** — это нормально на 8-м месяце 28-месячной стройки, но требует постоянного контроля по критическим путям (надземный монолит → кровля → фасад → отделка).
5. **Инженерные сети** (Мещеряков): 55 работ, все в статусе «не начато». Старт ожидается на этапе закрытия монолита.
6. **Наружные сети — силами Заказчика.** Это зона риска по координации: если Заказчик отстанет, ввод объекта в эксплуатацию (59 работ) сдвинется.
7. **Финансирование (аванс)** для одной из работ закреплено за Гончаровой А., но колонка «Факт Ф» = 0% при «План Ф» = 100% — стоит проверить, нет ли блокировок по авансам.
